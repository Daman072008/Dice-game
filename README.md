# Game of Luck

A simple command-line dice rolling game written in Python. This project demonstrates basic Python concepts such as modularity, loops, and user input handling.

## Files in this Project

* **`main.py`**: The entry point of the application. It handles the game loop and user interface.
* **`dice.py`**: Contains the logic for the dice roll using Python's `random` module.
* **`exit.py`**: Contains a simple function to display a thank-you message upon quitting.

## How to Run

1.  Ensure you have Python installed on your system.
2.  Keep all three files (`main.py`, `dice.py`, `exit.py`) in the same directory.
3.  Run the game using the following command:
    ```bash
    python main.py
    ```

## How to Play

* When the game starts, you will see a welcome message and two options.
* **Press [1]**: To "Roll the Dice" and see your lucky number (1-6).
* **Press [2]**: To exit the game.
* If you enter any other value, the game will notify you of an invalid input and ask again.
